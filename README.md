# vibe-composing

Original music compositions represented as text-based guitar and bass guitar tablatures, composed with the help of [Claude Code](https://claude.ai/code).

## About

This repository accompanies a talk about using Claude Code as a tool for music composition. Compositions are stored as plain text files using standard ASCII tab notation.

## Song

A rock ballad in C major. The entire song uses the chord progression C - Am - Dm - F - G (I - vi - ii - IV - V). Two bars per chord except F and G (one bar each), forming an 8-bar phrase in 4/4 time.
