# vibe-composing

A repository of original music compositions represented as text-based guitar, bass guitar, and ukulele tablatures.

## Project goal

Explore using Claude Code to assist with music composition.

## Song

A rock ballad in C major. The entire song uses the chord progression C - Am - Dm - F - G (I - vi - ii - IV - V). Two bars per chord except F and G (one bar each), forming an 8-bar phrase in 4/4 time.

## Format

Tablatures are stored as plain text files using standard tab notation. File names use an instrument prefix (guitar-, bass-, ukulele-). Fretboard diagrams use 20 frets for guitar and bass, 12 frets for ukulele. When adding, removing, or renaming files, update the Files section in README.md.
